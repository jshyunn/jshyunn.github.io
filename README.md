# jshyunn.github.io

취준 포트폴리오 겸 개인 블로그. 순수 HTML/CSS/JS로 작성했고 GitHub Pages로 바로 배포됩니다.

## 구조

- `index.html` — 포트폴리오 메인 페이지 (About / Skills / Projects / Contact)
- `css/style.css` — 스타일 (라이트/다크 모드 지원)
- `js/main.js` — 다크모드 토글, 모바일 메뉴 토글
